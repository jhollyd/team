Abstract

This thesis explores how dynamic user interface (UI) elements—specifically, animated micro-interactions—can enhance user engagement and emotional connection in web applications. Through a case study of an interactive hero section featuring a typing animation with randomized synonyms, we analyze the impact of motion, pacing, and personalization on user experience (UX). The study combines front-end development techniques with established UX principles, including Hick’s Law, cognitive load theory, and accessibility considerations. Findings suggest that purposeful animations improve dwell time and perceived interactivity, but must balance performance and inclusivity. The paper concludes with recommendations for future work, such as A/B testing and adaptive motion preferences.

Keywords: Web UI, User Experience (UX), Micro-interactions, Front-End Development, Animation

1. Introduction

1.1 Background

Static web interfaces often fail to capture user attention in an increasingly competitive digital landscape. Research shows that users form first impressions of a website within 50 milliseconds (Lindgaard et al., 2006), making the hero section a critical component for engagement. Dynamic UI elements, such as subtle animations and personalized content, can create emotional connections and guide user behavior.

1.2 Problem Statement

While animations are widely used in modern web design, many implementations prioritize aesthetics over usability. Poorly executed motion can increase cognitive load or exclude users with motion sensitivities. This thesis investigates:

  • How can micro-interactions (e.g., typing effects) improve engagement without compromising accessibility?

  • What technical strategies (e.g., preloading, fallbacks) ensure smooth performance?

  1. Literature Review: UX principles guiding dynamic UI design.

  4. Conclusion: Best practices and future directions.

Hick’s Law

  • Users take longer to decide when presented with more choices.

Cognitive Load Theory (Sweller, 1988)

  • Animations should aid comprehension. The typing effect mimics human communication, making the UI feel conversational.

Affordances & Signifiers
